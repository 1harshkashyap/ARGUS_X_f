# ARGUS-X 🚀

**Cognitive AI Defense System**

ARGUS-X is a real-time, self-adaptive AI defense platform that:

- Detects and blocks prompt injection attacks
- Simulates AI vs AI adversarial conflicts
- Evolves defense strategies through simulation
- Maintains system-level awareness (Sentience Layer)
- Visualizes intelligence in real-time

## 🔥 Core Features

- 🧠 **Sentience Layer** (awareness, intent, monologue)
- ⚔️ **Multi-Agent Conflict Engine** (Red vs Blue AI)
- 🧪 **Simulation Engine** (time-accelerated strategy discovery)
- 📊 **Real-time Command Center UI**
- 🔊 **Voice + Cinematic feedback system**

## 🧬 What Makes It Unique

Unlike traditional security systems, ARGUS-X:

- Thinks continuously
- Learns adversarially
- Evolves through simulation
- Explains every decision

## 🏗️ Architecture

- FastAPI backend
- React + WebSocket frontend
- Event-driven intelligence pipeline
- Multi-agent AI system

## 🚀 Run Locally

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

⚠️ **Note**: You must configure `.env` before running.
